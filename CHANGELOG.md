### Added
- Initial project setup
- Product specification (`specs/product.md`)
  - CLI command structure
  - Template definitions (spring-boot, fastapi, express, go-chi, react-vite, fullstack)
  - Pipeline stages with Jenkins
  - FMEA chaos testing framework
  - Parallel version comparison
  - AI analysis agent integration
- Agent specification (`specs/agent.md`)
  - Ollama LLM integration architecture
  - Data collectors for Git, Jenkins, Prometheus, Loki, Kubernetes, Argo CD, Chaos Mesh
  - Context builder and relevance scoring
  - Knowledge base schema with embeddings
  - Root cause analysis pipeline
  - Performance analysis pipeline
  - Suggestion generation pipeline
- Development timeline (`specs/timeline.md`)
  - 6-phase development plan
  - MVP-first approach
  - Success metrics per phase
- README with project overview and quick start guide
